2026-6月-02
以身体为重。

2026-6月-03
一天中的意外之喜，操场上举办小型的音乐节。本想回到寝室，走出图书馆不远听到操场似乎有歌声。内心经过一番挣扎后，我向操场走去。走近后，注意到是一个音乐节，结束后才知道专为今年毕业生举办的。24年时，我也来操场听过为那一年毕业生举办的音乐节，有种与过去的自己“见面”的感觉。

我很喜欢听音乐，尤其是一群人周围坐着，聆听着歌手的歌声。也正是因为如此，学校每年的十一、十二月份的大型音乐节我都会参加，除了上一年。那天晚上上课，应该请假。今年虽然在备考，但音乐节肯定是要听的，毕竟是最后一次。

今晚歌曲《鹿 be free》《晴天》《至少还有你》，还有一两首歌曲忘记了名字，唱得很不错，但之后的歌曲不如这几首。

虽然很喜欢听音乐，但近来我还是很少听它了。酷狗似乎在重复向我推荐听过的歌曲，有些歌听腻了，也就不想再听了。

2026-6月-05
在图书馆里注意到了一对情侣，我在他们后面，看样子也是要离开这里。我下意识的看向他们牵着的手，内心有向往，期待。听他们的说话语气，像是刚恋爱不久。他们之间的对话被我不经意的听到，并非什么有趣的笑话，却比笑话有感染力，以至于我也笑了起来。

我加快步伐，超过他们。不知缘由，他们玩起了石头剪刀布——一个规则简单，操作机械的游戏，却也能获得的欢乐。

2026-6月-12
为了制作 ppt，在 github 上找到 ppt-master 项目。也因为这个项目，第一次接触 skill, claude code, cedex等 AI 工具。也因此，这几天我对 L 站的公益站是非常的眼红，可惜要么需要邀请码，要么是5年的 github 账号。所幸 anyrouter 公益站可以通过教育邮箱注册。

使用 cc-switch 配置了 claude code 和 codex，我一开始不知道如何配置，出现各种错误，好在 rss阅读器 的 L 站订阅刚好更新 ccs + claude code的配置帖子，在尝试了不同的配置后，也终于解决这个问题 。不过 claude 总是 retry ，便选择了 codex。生成的 ppt 效果难以言说，ppt 最终还是亲手制作的。

2026-6月-15
下午得知晚上将举办26届毕业生的毕业典礼，想去看。但是有课，还要做 pre。等到下课后，往那里赶去，所幸还未结束。布置的聚光灯耀眼，我所在的位置很难看清唱歌的人，虽然只听了两首歌。其次，最后一个节目是模特走秀——绕着喷泉走到舞台上，只看了一会便走开了，其一因为灯光刺眼，其二是我嫉妒那些模特（男女都有）的身高，至少比我高很多，让我羡慕。如果回到过去，一定要好好吃饭。

2026-6月-19
端午节待在学校，没有回家，也没有学习，因为图书馆的内馆关闭一天。也算是一个让自己心安理得地放松的借口。

现在发觉时间过得好快，以前倒也发出类似的感叹。期末到了，同样代表一个学期的结束。最近是毕业季，我刚入学时，带领我报道的学长也在前几天离校。毕业后回到家是什么心情，我觉得应该类似我今天待在寝室的感受一样——无聊。反复打开 rss 阅读器等各种软件，期待着有什么新奇、特别的文章或各种东西可以引起注意，以度过茫然的时光。

相比之下，考研带来目标感，虽然累，但也是一种乐趣。我做不到每次学习都能进入心流，但是，用一年的时间尽自己最大努力完成某个目标，可以获得极大的满足感。这是在不考虑目标是否实现的情况下，也就是考研结束的那一天那一刻。从这一方面看，过程是大于结果的；当这一感觉散去或出成绩，也许就是结果大于过程了。

2026-6月-23
早上走向以往常去的窗口，我还未向食堂阿姨说吃什么，只见她已经拿好了三个鸡蛋。此时，我有错愕，也有高兴。她问我是不是还有一杯粥，我回没错。阿姨旁边的叔叔笑着说，他都记住了我每天早上吃三个鸡蛋和一杯粥。我听到这句话，也笑起来。

2026-6月-26
食堂阿姨仍然像询问别人一样询问我吃什么，没有像上次一样准备好鸡蛋。我觉得这样也挺好的，若每次都提前准备好，免不了会带来“压力”。
